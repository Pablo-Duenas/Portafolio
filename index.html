Pablo Dueñas Diez

Soy un desarrollador web apasionado y orientado a resultados, con experiencia en la creación de aplicaciones y sitios web modernos, eficientes y visualmente atractivos. Domino tecnologías como HTML, CSS, Bootstrap y TailwindCSS para construir interfaces intuitivas y responsivas, así como JavaScript y jQuery para desarrollar funcionalidades dinámicas y lógicas de negocio robustas.

Mi experiencia también incluye programación backend con PHP y bases de datos MySQL, así como el uso de Ajax para optimizar la comunicación asíncrona y mejorar la experiencia del usuario. Además, manejo control de versiones con GitHub, asegurando un flujo de trabajo colaborativo ordenado y seguro.
